Controle de Acesso LSI

O sistema de controle de acesso proposto leva em conta a existência de múltiplos usuários (professores), múltiplos ambientes (salas), restrições de horário (horários das aulas dos professores) e provê uma interface gerenciável por meio de uma aplicação web.

Cadastros

A aba CADASTROS compreende as páginas de inclusão de novos Agendamentos, Tags, Salas e Usuários ao banco de dados do sistema, permitindo seu gerenciamento e/ou interação aos demais módulos deste projeto.

Relatórios

A aba RELATÓRIOS proporciona aos usuários fichas de Pontualidade e Taxa de Frequência a partir de seus dados de interação ao objeto inteligente do sistema.

Registros

A aba REGISTROS disponibiliza páginas de gerenciamento de Agendamentos, Salas e Usuários previamente cadastrados no sistema proposto.
